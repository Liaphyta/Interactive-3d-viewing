In this lab assignment we are showing different models of shading and illumination.

The available models are shown below -

Gouraud/diffuse - Gouraud shading (vertex shading) with diffuse illumination.
Gouraud/specular - Gouraud shading (vertex shading) with the Phong illumination model to produce a specular highlight.
Phong/diffuse - Phong shading (pixel shading) with diffuse illumination.
Phong/specular - Phong shading (pixel shading) with the Phong illumination model to produce a specular highlight.

The user should be able to select the active lighting and shading models used by pressing the number keys between:
'u' (Gouraud/diffuse)
'i' (Gouraud/specular)
'o' (Phong/diffuse)
'p' (Phong/specular).

The user can interact with the shapes and with the light position.By clicking the 'L' button, the user will activate the movement of light.
By clicking the 'C' button it will activate the camera movement.
Translations (using arrow keys) -
">" (right): move the shape/light/camera to the right
"<" (left): move the shape/light/camera to the left
"/\" (up): move the shape/light/camera up
"\/" (down): move the shape/light/camera down
"," (comma): move the shape/light forward
"." (period): move the shape/light backward
